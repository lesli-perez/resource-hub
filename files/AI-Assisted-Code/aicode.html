Project Overview

This page includes an example submission for the AI Assisted Coding Assignment and an Assignment Page for students.

The original activity was based on the Hour of AI website. Since the online IDE environment was not accessible, the assignment was recreated as a standalone web project that does not require external tools to run.

Content

Prompt Engineering Comparison

Two different AI prompts were used to compare output quality and required debugging effort.

Simple Prompt

A basic instruction requested a Connections-style game using HTML, CSS, and JavaScript. The output provided a functional starting point but required significant refinement in layout, interaction design, and structure.

Detailed Prompt

A more advanced prompt included specific requirements such as UI structure, responsive layout behavior, and component organization. This version produced more complete and structured output with fewer required corrections.
